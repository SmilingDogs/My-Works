The Forkio

The example of fully adaptive and responsive site made to show content smoothly on all types of devices: PC, tablets, smartphones. Built with SASS/SCSS preprocessors, a bit of pure JS code for burger menu animation. Uses Gulp task-runner for combining different sections of SCSS and HTML in one file + optimization of file sizing.

Check this out!

The HAM Project

The example of static site made for PC/Desktops. Uses standard HTML, CSS (Flexbox, Grid technologies, pseudoelements, etc) and pure Javascript for animation of People Say section slider, Our Services tabs, for load button effects, etc. + uses Masonry flexible layout plug-in

Check this out!

The Railways

Another example of simple yet fully adaptive/responsive site for all devices. Built using HTML SCSS JS Gulp modules for different style,script,image tasks processing and file optimization.

Day and Night

This small project is an example of creative animation and transition effects made with simple CSS and some JS. For a day/night theme toggle, clouds and stars animation, just click on the Sun or Moon and watch a day turning into beautiful windy night!
